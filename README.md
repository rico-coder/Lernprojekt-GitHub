# Notenübersicht

## Projektbeschreibung
Mit dieser Java-Applikation kann ich Noten zu meinen jeweiligen Schulen erfassen.
Dabei wird in der jeweiligen Schule der Durchschnitt von jedem Fach berechnet.
Wiederum von den Durchschnitten wird der Gesamtdurchschnitt der Schule berechnet.
Die Noten der Schulen sollten über ein Eingabefeld zu einem gewählten Fach und der gewählten Schule hinzugefügt werden.
Dann wird automatisch der neue Notendurchschnitt berechnet.
Diese kann man danach natürlich auch wieder löschen.
Bei der Eingabe der Note wird eine Validierung überprüfen, ob die Note gültig ist.

---

## Funktionen
- Noten zu jedem Fach eintragen und löschen
- Durchschnitt pro Fach automatisch berechnen
- Gesamtdurchschnitt der Schule anzeigen
- Fehlerprüfung: Noten müssen zwischen 1 und 6 liegen
- GUI mit TableView und Buttons zur einfachen Bedienung
- Daten werden in einer JSON-Datei gespeichert und beim Start geladen (optional)
